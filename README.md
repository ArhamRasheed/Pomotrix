# Pomotrix ⏱️

A productivity app built around the **Pomodoro Technique** — stay focused, track your sessions, and get more done.

## 🔧 Tech Stack

- **Language:** TypeScript
- **Framework:** React
- **Styling:** Tailwind CSS

## 🚀 Features

- Customizable Pomodoro timer (work / short break / long break)
- Session tracking & history
- Clean, distraction-free UI
- Desktop notifications
- Keyboard shortcuts

## 📦 Getting Started

```bash
# Clone the repo
git clone https://github.com/ArhamRasheed/Pomotrix.git
cd Pomotrix

# Install dependencies
npm install

# Start the app
npm run dev
```

## 💡 About

The Pomodoro Technique breaks work into 25-minute focused sessions separated by short breaks, helping you maintain high productivity without burnout.
